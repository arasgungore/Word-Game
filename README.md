# word-game
A C++ project which lets you play the popular TV show Word Game (Kelime Oyunu) with 7000+ questions on the terminal to help you boost your word power.

## Run on terminal

```sh
g++ wordGame.cpp main.cpp -std=c++11 -o test
test
```
